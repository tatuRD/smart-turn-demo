マイクを許可して話すと、端末上だけでVADとSmart Turnを実行します。

アルゴリズム概要

  • VAD threshold: VAD probability がこの値以上なら ON、下回ると OFF とみなします。
  • Turn-taking threshold: Smart Turn の出力確率がこの値以上ならターンテイキング True、未満なら False と判定します。
  • Smart Turn 入力音声: マイクのモノラル音声を 16 kHz にリサンプリングして最大 8 秒ぶん保持し、発話終了後 200 ms で直近 8 秒を切り出して使います。短い場合は先頭をゼロ埋めし、その後 Whisper 互換の 80 x 800 log-Mel 特徴量に変換して推論します。出力確率がしきい値を超えて True になった場合は音声バッファを空にするため、同じ区間は次回の入力に含まれません。
